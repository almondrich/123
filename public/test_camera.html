Camera Test


Max file size: 5MB. Allowed formats: JPG, PNG, GIF, WebP


Expected Behavior

  • PC without webcam: Clicking "Open Camera" should show an error message about no camera device found
  • PC with webcam: Clicking "Open Camera" should request permission and show video feed
  • Mobile device: Should open the device camera
  • File Upload: Should accept image files and show preview
